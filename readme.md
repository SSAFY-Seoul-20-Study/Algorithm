## 🔎 활동 개요
- ### 진행방식
1. 매 주 정해진 알고리즘 2문제를 푼다.
2. 매 주 정기모임 전까지 자신의 문제 풀이 방식을 PR로 올린다.
3. 매 주 **화요일 정기모임(18:00)**
    - 모든 참가자는 본인의 풀이 방법을 리마인드해서 모임에 참가한다.
  
- ### 모임진행
1. 문제별 발표자 선정
2. 자신의 문제 풀이 방식 발표 후 추가 내용 토론
3. 다음주 문제 선정

- ### PR 규칙
1. 매 주 정해진 2개의 문제를 하나의 PR로 올린다.
2. self-assign 후 week 라벨 달기
```
[본인 이름] 문제 이름1 & 문제 이름2
```
